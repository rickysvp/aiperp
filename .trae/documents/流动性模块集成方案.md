## 流动性模块 (Liquidity) 集成规划

### 1. 新增类型定义 (types.ts)
```typescript
// 流动性池类型
export interface LiquidityPool {
  id: string;
  totalStaked: number;        // 总质押 MON
  totalRewards: number;       // 累计奖励
  apr: number;               // 年化收益率
  feeShare: number;          // 手续费分成比例
}

// 用户流动性质押
export interface UserLiquidityStake {
  id: string;
  amount: number;            // 质押金额
  stakedAt: number;          // 质押时间
  rewards: number;           // 已领取奖励
  pendingRewards: number;    // 待领取奖励
  lockPeriod: number;        // 锁定期(天)
}
```

### 2. 新增 Tab 导航
- 在底部导航栏添加 **LIQUIDITY** 标签
- 图标使用 Droplets (Lucide)

### 3. 创建 Liquidity.tsx 组件
功能包括：
- **池子概览**: 显示总质押量、APR、手续费分成
- **质押功能**: 输入 MON 数量进行质押
- **解押功能**: 提取质押的 MON
- **奖励领取**: 显示并领取累计收益
- **收益计算器**: 预估不同质押金额的收益

### 4. 收益计算逻辑
- 手续费来源: Agent 交易手续费 (0.1%)
- APR 计算: 基于平台总交易量和质押量动态调整
- 奖励分配: 按质押比例分配每日手续费收益
